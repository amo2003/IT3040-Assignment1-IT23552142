# IT3040 – ITPM Assignment 1 (Option 1)
## Transliteration Accuracy Testing (Singlish → Sinhala)

---

## 📌 Project Overview
This project automates testing of a Singlish to Sinhala transliteration system using Playwright.  
The system under test is:

https://www.pixelssuite.com/chat-translator

The goal is to identify incorrect transliterations and record results in an Excel file.

---

## 🛠️ Technologies Used
- Python  
- Playwright (Automation Testing)  
- OpenPyXL (Excel Handling)  

---

## ⚙️ Prerequisites
- Python 3.11 or 3.12  
- Google Chrome / Chromium browser  
- Internet connection
